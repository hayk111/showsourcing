Icons

By default icons inherit the font size name="product"

You can give a number in the [size] input name="product" [size]="24"

Or you can give one of our default size "s", "m" or "l" name="product" [size]="'s'" || [size]="'m'" || [size]="'l'"

Logos

Color

By default if a type is specified the color associated with the [type] will be used
type="product" | "supplier" | ... (use type color)

You can however override the color by specifying the [color] attribute
type="product" color="primary" || color="secondary" || color="vibrant"



Circle

type="product" [circle]="true" color="accent"
