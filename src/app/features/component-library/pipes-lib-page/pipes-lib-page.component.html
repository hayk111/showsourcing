Price

Input will be an object like this: {{price | json}} | appPrice

{{{ value: 55680, currency: "USD"} | appPrice}}

Or a number: 6000000 | appPrice

{{6000000 | appPrice}}

Const

Transform a const to it's given name or if any additional param is given the property named with that param

Input will be like this: 'input' | const: 'additional param 1': 'additional param 2'

If the input does not have second param, default will be name

'USD' | const: 'currency'

{{'USD'| const: 'currency'}}

'USD' | const: 'currency': 'symbol'

{{'USD' | const: 'currency': 'symbol'}}

Image

Display image based on an input that can be an object an array or a string

You can specify one of the [size] "xs", "s", "m", "l" or "xl" If you not set the size, default size will be "xl"

Object If the object is:

- AppImage, the pipe will find the first url in object.urls

[src]="AppImage | image: 's'" [src]="{{imageMock | image: 's'}}"

- PendingImage, the pipe will bypass security and trust the given value.data

[src]="PendingImage | image: 's'" [src]="this is trusted url"

- Supplier, product, Entity object... the pipe will find the first url in object.images

[src]="Supplier | image: 's'" [src]="{{supplierMock | image: 's'}}" String [src]="{{imageUrl}} | image" [src]="{{imageUrl | image}}" Array

The pipe will find the first url in object.urls of first item of the array

[src]=" "this is array of image object" | image" [src]="{{images | image}}"

Date to time ago

{{currentDate}} | timeAgo {{currentDate | timeAgo}}

File size

Transform a number to file size with its given precision (default is 2)

{{size}} | fileSize {{size | fileSize}} {{size}} | fileSize: 3 {{size | fileSize: 3}}

User name

Transform an object which has firstName and lastName

args -> true/false, default: false. Determine if we show the full name or just abbreviation

{{userData | json}} | formatUserName {{userData | formatUserName}} {{userData | json}} | formatUserName: true {{userData | formatUserName: true}}
