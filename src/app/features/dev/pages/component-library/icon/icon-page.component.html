Icons

By default icons inherit the font size name="product"

You can give a number in the [size] input name="product" [size]="24"

Or you can give one of our default size "s", "m" or "l" name="product" [size]="'s'" || [size]="'m'" || [size]="'l'"

Logos

Color

By default if a type is specified the color associated with the [type] will be used
type="product" | "supplier" | ... (use type color)

You can however override the color by specifying the [color] attribute
type="product" color="primary" || color="secondary" || color="vibrant"



Circle

type="product" [circle]="true" color="accent"



Circle

You can use the [circle] input to make it a circle type="product" | "comment" | "task" [circle]="true"



Size

You can specify one of the [size] "s", "m" or "l" type="product" size="s" | size="m" | size="l"

You can however select specific sizes for [backgroundSize] and [iconSize] type="product" [backgroundSize]="60" [iconSize]="35"



Image

If you specify a [logo] it will be used type="product" [logo]="imageMock"



Icon

You can set a specific icon icon="star" color="accent"



Initials

You can use initials logo: instead of the default icon it will display the initials of a name name="Supplier with name" type="supplier" size="m" | size="l" | size="xl"
